# Mission Control

A NextJS productivity dashboard for tracking projects, tasks, and system status.

## Features

- 📊 **Dashboard** - Overview of active projects and system stats
- ✅ **Task Manager** - Track todos with priorities
- 📝 **Quick Notes** - Rapid note-taking
- 🔗 **Quick Links** - Access to OpenClaw dashboard

## Getting Started

```bash
# Navigate to project
cd mission-control

# Install dependencies (already done)
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```

## Project Structure

- `src/app/page.tsx` - Dashboard
- `src/app/tasks/page.tsx` - Task manager
- `src/app/notes/page.tsx` - Quick notes
