# eBay Sample Listings

Ready-to-use templates. Replace text in [brackets] with your item details.

## Template 1: Electronics (Smartphone)

**Title Formula:**
```
[Brand] [Model] [Storage] [Carrier/Unlocked] [Color] [Condition] - Tested/Working
```

**Example Title:**
```
Apple iPhone 11 64GB Unlocked Black Good Condition - Tested Working
```

**Description:**
```
Apple iPhone 11 64GB Unlocked - Black

Condition: Good. Light scratches on screen, minor wear on corners. Fully functional. Battery health 85%.

What's Included:
- iPhone 11 64GB
- Original box (if available)
- Lightning cable

Testing:
- All buttons working
- Camera front/back tested
- Touch screen responsive
- Charging port working
- Face ID functional
- No iCloud lock

Shipping:
Ships within 1 business day via USPS Priority with tracking.

Returns:
30-day returns accepted. Item must be in same condition as received.

Questions? Message me!
```

**Pricing Strategy:**
- Check completed listings, price 5-10% below lowest similar
- Or: Start auction at 70% of expected value

## 🏷️ Title Optimization Tips

**ALWAYS Include:**
- Brand name
- Model number
- Key specifications (size, storage, etc.)
- Condition
- "Working/Tested" if applicable

**NEVER Include:**
- ALL CAPS (looks spammy)
- Symbols ($#@) — wastes title space
- "Look" or "L@@K"
- Fluffy words like "amazing" or "rare" (unless actually rare)

**Character Budget:**
- eBay gives 80 characters
- Use every character wisely
- Most important words first (mobile users see ~35 chars)

## 💰 Pricing Quick Reference

| Item Type | Research Method | Price Strategy |
|-----------|-----------------|---------------|
| Electronics | Completed listings (last 30 days) | Match lowest BIN or 10% under |
| Gaming | Check PriceCharting.com | Market price or auction |
| Cards | Sold listings only | Auction for high value, BIN for low |
| Tools | New price minus 40-60% | Slightly under market |
| Household | Completed listings | Auction if unsure, BIN if confident |

**BIN = Buy It Now**

## 🚢 Shipping Recommendations

| Item | Method | Cost Range |
|------|--------|------------|
| Small (cards, cables) | USPS First Class | $3-5 |
| Medium (phone, controller) | USPS Priority | $8-12 |
| Heavy (tools, console) | USPS Priority/FedEx | $15-25 |
| Bulky (mixer, furniture) | Calculated shipping | Varies by location |

Always use calculated shipping for items > 5 lbs.
