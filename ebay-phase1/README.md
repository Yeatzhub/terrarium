# eBay Phase 1 Revenue Package

Quick cash project to fund Phase 2 trading operations.

---

## 📂 Files Included

| File | Purpose |
|------|---------|
| `inventory_template.md` | Catalog template with high-value categories |
| `photo_checklist.md` | Complete photo setup guide |
| `sample_listings.md` | 5 ready-to-use listing templates |
| `shipping_guide.md` | Full shipping workflow + best practices |

---

## 🎯 Quick Start

1. **Inventory** → Walk through your space, fill out catalog (aim for 20 items)
2. **Research** → Check completed eBay listings for pricing
3. **Photos** → Set up near window, follow photo checklist
4. **Draft** → Use sample listings as starting point
5. **Ship** → Follow shipping guide when items sell

---

## 💰 Revenue Estimate

- **Target:** $200-500 (10-20 items @ $20-50 average)
- **Timeline:** List in 1 week, sell in 2-3 weeks
- **Net:** ~$150-400 after shipping/fees

## 🚀 Getting Started RIGHT NOW

**15-minute commitment:**

1. Walk to nearest closet/room
2. Identify 5 things you don't use
3. Check quick-win categories:
   - Electronics (old phones, cables, chargers)
   - Gaming (consoles, games, controllers)
   - Collectibles (cards, comics, coins)

**Write them down → You're started.**
