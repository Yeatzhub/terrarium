# Task: Mission Control Android App

Build a standalone Android app for Mission Control dashboard.

## Requirements

### Features
- Dashboard with agent status cards
- Task queue view (pending/completed)
- Quick task creation
- Real-time updates
- Dark theme (slate 950)

### Tech Stack
- Kotlin + Jetpack Compose
- MVVM + Clean Architecture  
- Room for local caching
- Retrofit for API calls
- Hilt for DI
- Material Design 3

### Screens
1. Dashboard - Overview + Quick Actions
2. Agents - Team status + task assignment
3. Tasks - Queue + history
4. Settings - API endpoints + preferences

### API Integration
- Connect to: http://100.125.198.70:3000/api
- Polling fallback (5 sec)
- Manual refresh

### Design
- Match Mission Control web design
- Dark background (slate-950)
- Cyan/purple accent colors
- Bottom navigation (Home, Agents, Tasks, Settings)

### Deliverables
- Complete Android Studio project
- Working APK for Pixel 8 Pro test
- README with build instructions

## Testing
Test on Pixel 8 Pro running Android 14
