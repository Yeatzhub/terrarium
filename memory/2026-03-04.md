### USB OTG Fix (Complete)

**Problem:** Mobian kernel sets `dr_mode = "peripheral"` in device tree, blocking host mode.

**Solution:** Created `enable-otg.sh` script that:
1. Backs up `boot_a` partition
2. Decompiles DTB, changes peripheral → host
3. Creates rollback script automatically
4. Only flashes slot A (safe recovery via slot B)

**Files:**
- `enable-otg.sh` - Main script
- `USB_OTG_FIX.md` - Documentation
- `FIX_APPROACHES.md` - All fix roadmaps
- `KNOWN_ISSUES.md` - Issue breakdown

**Kernel Sources:**
- Mobian: `kernel-src/` (6.12.74, cloned from mobian/qcom-linux)
- Nethunter: `nethunter-kernel/` (4.x, cloned from acai66/op6-op6t-nethunter-kernel)

**Key DTB file:** `sdm845-oneplus-fajita.dtb`
