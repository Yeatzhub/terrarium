### Known Issues — Research Findings (2026-03-22)

**GitHub Issue #854** (opened Jan 12, 2026):
- OnePlus 6T (`fajita`) reboots to fastboot instead of booting
- **Status:** UNRESOLVED — no fix available yet
- Affects both `hello` example and default configs

**Root Cause:**
- SDM845 mainline kernel is outdated (last updated ~2 years ago)
- Kernel bugs: EEPROM not read properly → restricted WiFi channels, random MAC
- See also Issue #855 (Mainline 845 SoC feature-set)

**Working Alternatives:**
1. **postmarketOS** — Confirmed working on fajita (many users report success)
2. **OnePlus 6 (enchilada)** — Better Mobile NixOS support than 6T
3. **LineageOS** — Android 11/12 works well

**iXit Blog (Dec 2025):**
- Active effort to upstream SDM845 fixes
- OnePlus 6/6T both noted as "hopefully everything works"
- May resolve boot issues once patches land upstream

**Strategy (2026-03-22):**
1. Try Mobile NixOS flash — may work despite issue
2. **If boot fails → pivot to Mobian** (not postmarketOS)
3. postmarketOS lacks Ollama support; Mobian is Debian-based with full package ecosystem

**Mobian Advantages:**
- Debian-based → Ollama available via standard packages
- Good mobile support (GNOME/Phosh)
- Active development for SDM845 devices
