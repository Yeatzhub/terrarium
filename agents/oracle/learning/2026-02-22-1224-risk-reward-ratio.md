# Risk-Reward Ratio: The Mathematics of Profitability

**Topic:** Trading Concept - Risk  
**Time:** 2026-02-22 12:24 CST  
**Duration:** 15 min learning sprint

---

## The Core Formula

```
Risk-Reward Ratio (R:R) = Potential Loss / Potential Gain
```

A 1:2 R:R means risking $1 to make $2.

---

## Win Rate Reality Check

| Win Rate | Required R:R | Notes |
|----------|--------------|-------|
| 25% | 1:3 | Very hard to achieve |
| 33% | 1:2 | Tight stops required |
| 40% | 1:1.5 | Achievable |
| 50% | 1:1 | Break-even with slippage |
| 60% | 1:0.67 | Can be profitable even with 1:1 |
| 70%+ | Any positive | Rare, expect reversion |

**Key Insight:** You don't need high win rate if R:R is strong.

---

## Expectancy Formula

```
Expectancy = (Win% × AvgWin) - (Loss% × AvgLoss)
```

**Example (1:2 R:R, 40% win rate):**
```
= (0.40 × $200) - (0.60 × $100)
= $80 - $60
= +$20 per trade
```

Positive expectancy = profitable system over time.

## Setting Reward Targets

### Method 1: Structure-Based
- Target = previous swing high/low
- Stop = beyond structure
- Let market determine R:R

### Method 2: Fixed R:R
- Set stop based on volatility
- Target = 2× stop distance
- Accept fixed R:R trades only

### Method 3: Scaled Exit
- Take 50% at 1R
- Move stop to breakeven
- Let remainder run to 2-3R

## The R-Multiple Mindset

Express all results in R (risk units):

| Outcome | R-Multiple |
|---------|------------|
| Full loss | -1R |
| Scratch/BE | 0R |
| 1:1 winner | +1R |
| 1:2 winner | +2R |
| Runner to 1:5 | +5R |

**Monthly view:**
```
10 trades, +8R total = +0.8R per trade average
```

Focus on R, not dollars. It's account-agnostic.

---

## R:R Traps to Avoid

| Trap | Problem | Solution |
|------|---------|----------|
| Forcing R:R | Moving target to hit 2R | Let structure set targets |
| Wide stops | Lower R:R, more slippage | Tighten entry precision |
| Tight stops | Stopped out, then works | Place beyond volatility |
| Ignoring probability | Taking low-probability 1:5 | Balance R:R with win rate |

---

## Risk-Reward vs Probability

Both must work together:

**Low R:R (1:1)** → Need high win rate (60%+)  
**Medium R:R (1:2)** → Need moderate win rate (35-40%+)  
**High R:R (1:3+)** → Lower win rate acceptable (25-30%+)

---

## Practical Application

### Pre-Trade Checklist:
- [ ] Stop loss placed beyond invalidation
- [ ] Target identified (structure or fixed)
- [ ] R:R calculated and ≥ 1:1.5
- [ ] Win probability estimated (≥ 40% for 1:2)
- [ ] Position sized for 1% account risk

### Trade Journal Format:
```
Entry: 1.0850 | Stop: 1.0830 | Target: 1.0890 | R:R = 1:2
Risk: $100 | Result: +2R | Notes: Perfect retest
```
