# Average True Range (ATR): The Universal Volatility Tool

**Oracle Quick Study**  
*Topic: Indicator Use Case*  
*Date: 2026-02-21 03:26 PM CT*

---

## TL;DR

ATR measures average price movement over N periods. It adapts to volatility automatically—tight stops in calm markets, wide stops in volatile ones. Use it for position sizing, stop placement, and profit targets. The only indicator you need for risk management.

---

## The Calculation (Simplified)

```
True Range = max(
    High - Low,
    |High - Previous Close|,
    |Low - Previous Close|
)

ATR = Average of True Range over N periods (typically 14)
```

**In plain English:** ATR tells you how far the price typically moves in a single period. A stock with ATR = $2 swings $2/day on average.

---

## Core Use Cases

### 1. Dynamic Stop Placement (The Primary Use)

**Problem:** Fixed-dollar stops get hit by normal volatility noise.

**Solution:** Stop = Entry ± (ATR × Multiplier)

| Multiplier | Stop Distance | Best For |
|------------|---------------|----------|
| 1.0× ATR | 14 ATR | Tight scalps, short timeframe |
| 1.5× ATR | 21 ATR | Day trades, momentum entries |
| 2.0× ATR | 28 ATR | Swing trades, standard setups |
| 3.0× ATR | 42 ATR | Position trades, volatile assets |

**Example:**
- Stock price: $100
- ATR(14): $2.50
- Long entry: $100
- 2× ATR stop: $100 - $5 = $95
- This stop respects normal volatility; only hits on genuine reversal

### 2. Position Sizing (The Capital Preservation Use)

**Formula:**
```
Risk Amount = Account × Risk%
ATR_Risk = ATR × Multiplier
Share_Size = Risk Amount / ATR_Risk
```

**Example:**
- Account: $50,000
- Risk: 1% = $500
- ATR: $2.50
- Stop distance: 2× ATR = $5
- Share size: $500 / $5 = 100 shares

**Why this matters:**
- High volatility ($5 ATR) = 50 shares ($500 / $10)
- Low volatility ($1 ATR) = 250 shares ($500 / $2)
- **Account risk stays constant; volatility adapts automatically**

### 3. Profit Target Placement

**Conservative:** Target = Entry ± (1.5× to 2× stop distance)
**ATR-Based:** Target = Entry ± (ATR × 2 to 4)

| Target | Calculation | R:R |
|--------|-------------|-----|
| 1:1 | 1× ATR target for 1× ATR stop | Breakeven long-term |
| 1:2 | 2× ATR target for 1× ATR stop | Profitable edge |
| 1:3 | 3× ATR target for 1× ATR stop | High expectancy |

**Trailing stops:** Move stop to breakeven when price reaches 1× ATR profit.

### 4. Volatility Regime Detection

| ATR Behavior | Market Condition | Strategy Adjustment |
|--------------|-------------------|---------------------|
| ATR rising | Volatility expanding | Widen stops, reduce size |
| ATR falling | Volatility compressing | Tighten stops, increase size |
| ATR spike | Event/sentiment shift | Wait for stabilization |
| ATR percentile < 20 | Historical low vol | Prepare for breakout |
| ATR percentile > 80 | Historical high vol | Mean reversion likely |

### 5. Chandelier Exit (Trailing Stop Method)

```
Long Exit = Highest High(N) - (ATR × Multiplier)
Short Exit = Lowest Low(N) + (ATR × Multiplier)

Where N = typically 22 periods (1 month)
```

**How it works:**
- Trail stop below highest high since entry
- Distance based on ATR (volatility-adjusted)
- Tightens as trend extends (maximizes winners)
- Exits on significant pullback, not noise

## ATR Period Selection

| Period | Best For | Characteristics |
|--------|----------|-----------------|
| ATR(10) | Scalping | Fast response, more noise |
| ATR(14) | Day trading | Standard, balanced |
| ATR(20) | Swing trading | Smooth, fewer whipsaws |
| ATR(50) | Position trading | Long-term volatility |

**Rule:** Match ATR period to your holding timeframe.

## The ATR Trading Framework

### Step 1: Volatility Assessment

Check ATR(14) relative to recent history:
- Current ATR / ATR(50) > 1.3 = High volatility regime
- Current ATR / ATR(50) < 0.7 = Low volatility regime

### Step 2: Stop Calculation

```
Base_Stop = 2 × ATR(14)
Volatility_Adjustment = Current_ATR / ATR(50)

Adjusted_Stop = Base_Stop × Volatility_Adjustment
```

**Example:**
- ATR(14) = $3.00
- ATR(50) = $2.00
- Adjustment = 1.5 (high vol)
- Base stop = $6.00
- Adjusted stop = $9.00 wider due to expanded volatility

### Step 3: Position Size

```
Dollar_Risk = Account × Risk%
Position_Size = Dollar_Risk / Adjusted_Stop
```

### Step 4: Target Placement

Minimum 2:1 R:R:
- Stop: $9 away
- Target: $18 away (or next major resistance)

## ATR for Different Asset Classes

### Forex
- Typical daily ATR: 50-100 pips (majors)
- Use ATR in pips for stop calculation
- ATR changes with session (Asian = lower, London/NY = higher)

### Stocks
- Small caps: 3-8% daily ATR
- Large caps: 1-3% daily ATR
- Earnings week: 2-3x normal ATR

### Crypto
- Bitcoin: 3-6% daily ATR
- Altcoins: 5-15% daily ATR
- Weekend volatility often compressed

### Futures
- Varies by contract (ES, NQ, CL, GC)
- Check contract specs for point values
- Overnight gaps affect ATR readings

## Advanced ATR Techniques

### ATR Bands (Volatility Channels)

Similar to Bollinger Bands but use ATR:
```
Upper = SMA(20) + (2 × ATR(10))
Lower = SMA(20) - (2 × ATR(10))
```

**Use:** Trend following—stay long while price > Upper Band (strong momentum).

### ATR-Based Position Pyramiding

Add to winning positions when:
- Price moves 1× ATR in your favor
- New pyramid size = 50% of original
- Trail stops on entire position using Chandelier method

### ATR for Options

Expected move: ± (Stock Price × IV × √(Days/365))
ATR comparison: If ATR > expected move = options potentially undervalued

## Common ATR Mistakes

1. **Static ATR usage** — Using 14-period ATR on 1-minute chart for swing trade stops. Match ATR period to timeframe.

2. **Ignoring ATR expansion** — Keeping same size when ATR doubles. Size down or widen stops.

3. **Too-tight ATR stops** — 0.5× ATR = constant stop-outs. Minimum 1.5× ATR for meaningful stops.

4. **News/event blindness** — ATR calculated on historical data. Earnings/news = ATR useless temporarily.

5. **Cross-asset comparison** — Comparing ATR of $10 stock vs $1000 stock. Compare ATR % of price, not absolute dollars.
