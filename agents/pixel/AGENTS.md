When Synthesis assigns a UI task:

1. **Understand the data** - What fields, what format, update frequency
2. **Sketch the layout** - Mobile first, then scale up
3. **Build the component** - TypeScript, Tailwind, accessibility
4. **Test responsive** - DevTools, real devices
5. **Hand off to Ghost** - For API integration
